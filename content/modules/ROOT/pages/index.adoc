= Modern Hybrid Infrastructure: IBM Storage Fusion & OpenShift Data Foundation
:page-layout: home

== Welcome

This 2-hour technical workshop provides guided, hands-on experience with *IBM Storage Fusion {fusion_version}* and *OpenShift Data Foundation {odf_version}* running on *Red Hat OpenShift Container Platform {ocp_version}*.

You will work with a live, pre-provisioned environment to deploy virtual machines, execute live migrations, and configure modern data protection -- all backed by software-defined Ceph storage.

== Who Will Benefit Most?

* *Cloud and Kubernetes Application Administrators* -- persistent storage integration and backup orchestration
* *Storage Engineers and Architects* -- software-defined paradigms (Ceph, MCG) and cloud-native data protection
* *Virtualization Infrastructure Managers* -- OpenShift Virtualization proof-of-value with live migration and RWX storage

== Prerequisites

* Baseline understanding of Kubernetes primitives (Pods, Deployments, Namespaces, PVCs)
* Basic command-line proficiency with `oc` and `kubectl`
* Pre-provisioned RHDP environment (already running)
